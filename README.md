# Wisdom-Quick
A simple weather application with a modern UI built using:  - HTML - Tailwind CSS via CDN - JavaScript - `async/await` - Open-Meteo API

## Features

- Search for the current weather in any city
- Fetch data asynchronously using `async/await`
- Display temperature, feels-like temperature, humidity, precipitation, and wind speed
- Weather icons that reflect the current conditions
- Built-in `Dark Mode`
- Clear loading and error states

## How to Run

1. Open `index.html` in your browser.
2. Enter a city name.
3. Click the `Search` button.

## APIs Used

- Geocoding:
  `https://geocoding-api.open-meteo.com/v1/search`
- Forecast:
  `https://api.open-meteo.com/v1/forecast`
